Orvia Hydration Indication Patch

A simple wearable patch that changes colour depending on hydration levels. The Orvia patch helps athletes, carers and everyday users recognise dehydration before it becomes dangerous.

Product Features

Sweat Detection

The patch absorbs sweat and detects moisture levels during activity.

Colour Indicator

The indicator layer changes colour to visually show hydration levels.

Low Cost

The patch is affordable, making hydration monitoring accessible.

How It Works

The Orvia Hydration Indication Patch contains a moisture sensitive layer that reacts with sweat. As sweat increases the indicator changes colour to warn the user they may need to drink water.

1. Apply

Place the patch on the skin before exercise or outdoor activity.

2. Monitor

The patch absorbs sweat and monitors hydration conditions.

3. Alert

The colour change signals when the user should drink water.

Orvia is a student innovation project developed for the Junior Achievement programme. The aim is to design an affordable and simple hydration monitoring solution that helps people stay healthy and safe during activity.
